Basico
- Achar usuários usando o modulo re (expressoes regulares)
- Mensagens customizadas para cada usuario
- Mandar imagens, arquivos, links
- Receber inputs do usuario para escolher pessoas e mensagens especificos, diretamente no terminal
- Conectar as minhas variaveis PATH para acesso rápido

Intermediario
- Reutilizar um driver aberto para colocar novos inputs, 
basicamente não ter que abrir um driver novo para cada vez que rodar o programa
- Coletar o nome de todas as pessoas em um grupo
- Modularizar mais o código
- Implementar conceitos de Orientação a Objetos ou Programação Funcional

Avançado
- Interface simples com opções de funcionalidades básicas e intermediarias
